# Match Recording → Scouting Pipeline

Description: Systematically record tournament matches with tools like SwingVision, extract patterns, and generate a coaching brief before the next tournament. The highlight reel is a byproduct that feeds the scouting workflow. This is the highest-leverage thing after the app because it directly attacks the visibility problem — athletes currently have no structured video evidence when applying to colleges.
Effort: Operational more than technical — deploy existing tools, build the workflow.
Potential: Proves the core thesis. One athlete with a real profile is the case study.
Status: Backlog
Why it matters: US college coaches recruit on video. Mexican juniors don't have it in any organized form.
