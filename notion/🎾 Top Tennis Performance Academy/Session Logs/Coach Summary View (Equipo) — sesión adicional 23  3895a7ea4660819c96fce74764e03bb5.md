Status: Complete
What we did: Sesión adicional del 23 Jun. Se diseñó y construyó de punta a punta la vista Equipo para coaches. Primero se discutió el diseño: cards mobile-first como layout principal con toggle a tabla, ordenado por UTR descendente, tabs 'Mis atletas / Equipo' en la sección de Atletas. Se construyó un mock visual interactivo en el chat para validar la idea antes de codear. Después se implementó todo: nuevo Equipo.jsx con query batched de atletas + coaches + reportes + métricas, cards con borde cobre para atletas propios y badge 'Ver' para ajenos, pills de últimos 3 reportes por atleta con dot verde/gris según completado. Se actualizó AlumnoDetalle.jsx para modo solo lectura cuando el coach no es el dueño del atleta (oculta botón de nuevo reporte, muestra 'Solo lectura'). Se aplicó una migración en Supabase con 4 políticas RLS SELECT para lectura cruzada de reportes entre coaches, sin tocar permisos de escritura. Lint y tests pasaron sin errores, commit hecho.
